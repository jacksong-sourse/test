# SKILL - Test

[![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](https://www.apache.org/licenses/LICENSE-2.0)

> 300 名性格各异的虚拟用户，给你的软件打分找 Bug

Test 是一个 TRAE Skill，模拟 300 名性格完全不同的虚拟用户对软件产品进行全方位体验测试。每个用户从自己的视角给出 ⭐1.0-5.0 的应用商店式评分，并可能发现 Bug。

## 使用

在 TRAE 对话中输入：

```
帮我用 Test 测试一下 http://localhost:3000
```

## 30 种用户性格

300 名用户均匀分配至 30 种性格类型，每种约 10 人：

| 性格类型 | 代号 | 典型特征 |
|---------|------|---------|
| 完美主义者 | 细节控 | 追求极致，一像素不对都不行 |
| 暴躁老哥 | 急性子 | 稍微卡顿就开骂，零容忍 |
| 技术宅 | 极客 | 喜欢挖深层逻辑，善于发现隐藏问题 |
| 小白用户 | 小白 | 不太懂技术，容易迷路，但直觉敏锐 |
| 安全专家 | 白帽 | 专攻安全漏洞，XSS/SQL注入/权限绕过 |
| 性能狂魔 | 测速人 | 超过 1 秒就烦躁，必测极限并发 |
| 设计师 | 审美癌 | 视觉第一，交互必须丝滑 |
| 产品经理 | 逻辑怪 | 流程不合理必指出来，需求思维 |
| 老年用户 | 慢节奏 | 字太小看不清，按钮太小点不到 |
| 残障用户 | 无障碍 | 关注可访问性、屏幕阅读器、键盘导航 |
| 外籍用户 | 语言敏感 | 关注国际化、翻译、文化适配 |
| 佛系用户 | 无所谓 | 能用就行，很少抱怨 |
| ... | ... | 共 30 种 |

## 输出

测试完成后输出完整报告：

- **⭐ 综合评分**：类应用商店的 1.0-5.0 总分（300 人评分均值）
- **📊 评分分布**：5 星到 1 星的人数分布直方图
- **👥 按性格的评分分群**：不同性格类型的评分差异
- **🐛 Bug 清单**：按 P0-P4 严重等级排列，含复现步骤和修复建议
- **💬 精选用户原声**：20 条最具代表性的用户评价

```
╔══════════════════════════════════════╗
║     🏆 Test 用户评测报告              ║
║        ⭐ 3.8 / 5.0                  ║
║     🐛 发现 47 个 Bug                 ║
╚══════════════════════════════════════╝
```
